## Summary
A Node.js module to help you get insight into your package managing

## Installation
```bash
$ npm install
```

## Build
```bash
$ npm run build
```

## Tests
```bash
$ npm test
$ npm run test-html # Generate HTML coverage report
```

## Docs
```bash
$ npm run docs
```
